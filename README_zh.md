# 说明

lv_100ask_freetype 由[百问网](https://www.100ask.net)推出，旨在提供一个开箱即用的 freeType 库。

支持的平台：

- Espressif: ESP32、ESP32-S2、ESP32-S3
- Windows：win32、win64

# 获取仓库

```shell
git clone ...
git submodule update --init --recursive
```
